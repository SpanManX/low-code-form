## 项目目录结构
```
src/
├── assets
│     └── svg（存放SVG文件）
│     └── templates （存放模板文件）
│       └── components（vue 模板组件）
│
├── components （vue 组件）
│
├── demo
│
├── store （跨文件数据共享）
│
├── utils （存放封装的函数）
│
└── view （存放vue文件）
```
